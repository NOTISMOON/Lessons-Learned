## 一、直接引入 CSS 文件

### 基本用法：

```css
/* styles.css */
.button {
  background: blue;
  color: white;
}
import './styles.css';

function Component() {
  return <button className="button">Click</button>;
}
```

### 问题：

- 

  **全局污染**：类名全局生效，容易冲突

- 

  **难以维护**：项目大了后难以追踪样式影响范围

- 

  **相当于 Vue 没有 scoped**

### 适用场景：

- 

  小型项目

- 

  全局样式、重置样式

- 

  第三方样式库

## 二、CSS Modules（推荐）

### 配置要求：

Create React App 默认支持，无需额外配置

### 基本用法：

```css
/* Button.module.css */
.button {
  padding: 10px 20px;
  background: #007bff;
  color: white;
  border-radius: 4px;
}

.primary {
  background: #1890ff;
}

.large {
  padding: 15px 30px;
  font-size: 16px;
}
import styles from './Button.module.css';

function Button({ type, size }) {
  // 方法1：直接访问
  return (
    <button 
      className={`${styles.button} ${styles[type]} ${styles[size]}`}
    >
      Click
    </button>
  );
  
  // 方法2：动态组合
  const buttonClasses = [
    styles.button,
    type && styles[type],
    size && styles[size]
  ].filter(Boolean).join(' ');
  
  return <button className={buttonClasses}>Click</button>;
}
```

### 编译后效果：

```html
<!-- 编译后的类名是唯一的 -->
<button class="Button_button__abc123 Button_primary__def456">Click</button>
```

### 优点：

1. 

   **局部作用域**：自动添加 hash，避免冲突

2. 

   **类型安全**：TypeScript 支持良好

3. 

   **可维护**：样式与组件在同一目录

4. 

   **支持组合**：可以继承和组合样式

### 高级用法 - 组合样式：

```css
/* 组合示例 */
.base {
  padding: 10px;
  border: 1px solid #ccc;
}

.primary {
  composes: base;
  background: blue;
  color: white;
}
```

## 三、使用 classnames 库（CSS Modules 增强）

### 安装：

```bash
npm install classnames
# 或
yarn add classnames
```

### 基本用法：

```jsx
import classNames from 'classnames';
import styles from './Button.module.css';

function Button({ primary, disabled, size }) {
  const buttonClass = classNames(
    styles.button,                    // 始终存在
    {
      [styles.primary]: primary,     // 条件添加
      [styles.disabled]: disabled,
      [styles.large]: size === 'large',
      [styles.small]: size === 'small'
    }
  );
  
  return <button className={buttonClass}>Click</button>;
}
```

### 绑定模式（简化写法）：

```jsx
// 创建绑定函数
import classNames from 'classnames/bind';
import styles from './Button.module.css';

const cx = classNames.bind(styles);

function Button({ variant, outline }) {
  return (
    <button className={cx('button', variant, { outline })}>
      Click
    </button>
  );
}
```

### 优点：

1. 

   **条件类名**：逻辑更清晰

2. 

   **动态类名**：方便条件渲染

3. 

   **链式调用**：支持多种语法

## 四、Styled-components（CSS-in-JS）

### 安装：

```bash
npm install styled-components
# 或
yarn add styled-components
```

### 基本用法：

```jsx
import styled from 'styled-components';

// 创建样式化组件
const StyledButton = styled.button`
  padding: ${props => props.size === 'large' ? '15px 30px' : '10px 20px'};
  background: ${props => props.primary ? '#007bff' : '#6c757d'};
  color: white;
  border: none;
  border-radius: 4px;
  font-size: ${props => props.size === 'large' ? '16px' : '14px'};
  cursor: pointer;
  transition: background 0.3s;
  
  &:hover {
    background: ${props => props.primary ? '#0056b3' : '#545b62'};
  }
  
  &:disabled {
    opacity: 0.6;
    cursor: not-allowed;
  }
`;

// 使用
function App() {
  return (
    <>
      <StyledButton>Default</StyledButton>
      <StyledButton primary>Primary</StyledButton>
      <StyledButton primary size="large">Large Primary</StyledButton>
    </>
  );
}
```

### 扩展样式：

```jsx
// 基于现有组件扩展
const SuccessButton = styled(StyledButton)`
  background: #28a745;
  
  &:hover {
    background: #1e7e34;
  }
`;

// 传递 props
const IconButton = styled.button`
  display: inline-flex;
  align-items: center;
  gap: 8px;
  
  &::before {
    content: "${props => props.icon || '⚡'}";
  }
`;
```

### 主题支持：

```jsx
import { ThemeProvider } from 'styled-components';

const theme = {
  colors: {
    primary: '#007bff',
    success: '#28a745',
    danger: '#dc3545'
  }
};

const ThemedButton = styled.button`
  background: ${props => props.theme.colors.primary};
`;

<ThemeProvider theme={theme}>
  <ThemedButton>Themed</ThemedButton>
</ThemeProvider>
```
